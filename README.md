# Boss Killer

## 게임 컨셉
테라리아와 같은 횡스크롤 RPG 게임에서 보스전만을 채용한 액션 게임  
파밍과 같은 RPG요소 배제

## 개발범위
- 최소 둘 이상의 보스 몬스터  
-> 보스 행동양식
- 캐릭터 조작
- UI
- 리소스 수집 및 제작
- 스테이지 구성

## 예상 실행 흐름
### 타이틀 -> 첫 보스 대면 씬 -> 전투 -> 보스 엔딩 씬 -> 두번째 보스 대면 씬 ... -> 게임 엔딩
