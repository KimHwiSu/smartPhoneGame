# Boss Killer

## 게임 컨셉
테라리아의 보스전만을 채용한 액션 게임  
파밍과 같은 RPG요소 배제

## 개발범위
- 보스 행동양식
- 캐릭터 조작
- UI
- 리소스 수집 및 제작
- 스테이지 구성
